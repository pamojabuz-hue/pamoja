# PAMOJA Mobile App

Flutter mobile application for PAMOJA - The Business Operating System

Beautiful, intuitive interface for POS operations on the go.

## Features

✅ **Dashboard** - Real-time business metrics
✅ **POS Interface** - Easy product selection and checkout
✅ **Inventory Tracking** - Stock management
✅ **Profile** - User account management
✅ **Offline Support** - Works without internet
✅ **Receipt Printing** - Wireless printer support

## Setup

### Prerequisites
- Flutter SDK v3+
- Dart
- iOS or Android device/emulator

### Installation

```bash
flutter pub get
```

## Running

### Android
```bash
flutter run
```

### iOS
```bash
flutter run -d iPhone
```

## Building

### Android APK
```bash
flutter build apk
```

### iOS IPA
```bash
flutter build ios
```

## Tech Stack

- Flutter 3+
- Provider for state management
- SQLite for local storage
- HTTP for API calls

## Screens

- **Dashboard** - Key metrics and recent transactions
- **POS** - Product grid with cart management
- **Inventory** - Stock levels with status
- **Profile** - User account and settings

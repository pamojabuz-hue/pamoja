# PAMOJA Web Dashboard

React-based web dashboard for PAMOJA - The Business Operating System

Beautiful, responsive interface for managing your business operations.

## Features

✅ **Dashboard** - Real-time business metrics and charts
✅ **POS System** - Easy point-of-sale interface with cart management
✅ **Products** - Manage your product catalog
✅ **Inventory** - Track stock levels with alerts
✅ **Invoices** - Create and manage customer invoices
✅ **Reports** - Comprehensive analytics and reports

## Setup

```bash
npm install
```

## Running

### Development
```bash
npm start
```

Opens http://localhost:3000

### Build for Production
```bash
npm run build
```

## Tech Stack

- React 18
- Redux Toolkit for state management
- Tailwind CSS for styling
- Recharts for data visualization
- React Icons for beautiful icons

## Available Pages

- `/` - Dashboard with key metrics
- `/pos` - Point of sale interface
- `/products` - Product management
- `/inventory` - Inventory tracking
- `/invoices` - Invoice management
- `/reports` - Analytics & reports
